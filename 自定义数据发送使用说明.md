# 自定义数据发送使用说明

## 功能概述

新增了4个自定义数据发送函数，可以发送任意长度和内容的数据给下位机，自动完成打包（帧头、长度、CRC32校验）。

## 函数接口

### 1. 发送字节数组（vector版本）

```cpp
void sendCustomData(const std::vector<uint8_t>& data);
```

**使用示例：**
```cpp
// 发送自定义字节数据
std::vector<uint8_t> my_data = {0x01, 0x02, 0x03, 0x04, 0x05};
sendCustomData(my_data);
```

### 2. 发送字节数组（指针版本）

```cpp
void sendCustomData(const uint8_t* data, size_t length);
```

**使用示例：**
```cpp
// 发送C风格数组
uint8_t my_data[] = {0x01, 0x02, 0x03, 0x04};
sendCustomData(my_data, sizeof(my_data));

// 或者动态构建
uint8_t buffer[100];
size_t offset = 0;
buffer[offset++] = 0x01;  // 添加一个字节
// ... 添加更多数据
sendCustomData(buffer, offset);
```

### 3. 发送浮点数数组（vector版本）

```cpp
void sendFloatArray(const std::vector<float>& floats);
```

**使用示例：**
```cpp
// 发送浮点数数组
std::vector<float> data = {1.23f, 4.56f, 7.89f, 10.11f};
sendFloatArray(data);
```

### 4. 发送浮点数数组（指针版本）

```cpp
void sendFloatArray(const float* floats, size_t count);
```

**使用示例：**
```cpp
// 发送C风格浮点数组
float data[] = {1.23f, 4.56f, 7.89f};
sendFloatArray(data, 3);
```

## 数据包格式

所有函数自动打包成以下格式：

```
[0xFF][0xAA][长度(2字节)][数据(N字节)][CRC32(4字节)]
```

| 字段 | 大小 | 说明 |
|------|------|------|
| 帧头 | 2字节 | 固定为 0xFF 0xAA |
| 长度 | 2字节 | 数据段长度（小端序） |
| 数据 | N字节 | 你的自定义数据 |
| CRC32 | 4字节 | 校验值（小端序） |

**总包大小 = 8 + N 字节**

## 实际使用示例

### 示例1：在 write() 函数中发送自定义数据

```cpp
hardware_interface::return_type TideHardwareInterface::write(
    const rclcpp::Time& /*time*/,
    const rclcpp::Duration& /*period*/)
{
  // 方式1：发送浮点数数组
  std::vector<float> data = {
    static_cast<float>(state_positions_[0]),  // 关节1位置
    static_cast<float>(state_velocities_[0]), // 关节1速度
    static_cast<float>(state_positions_[1]),  // 关节2位置
    static_cast<float>(state_velocities_[1])  // 关节2速度
  };
  sendFloatArray(data);
  
  // 方式2：发送混合数据
  std::vector<uint8_t> custom_data;
  custom_data.push_back(0x01);  // 数据类型标识
  
  // 添加浮点数
  float value1 = 123.45f;
  uint8_t* ptr = reinterpret_cast<uint8_t*>(&value1);
  custom_data.insert(custom_data.end(), ptr, ptr + sizeof(float));
  
  // 添加整数
  uint32_t value2 = 12345;
  ptr = reinterpret_cast<uint8_t*>(&value2);
  custom_data.insert(custom_data.end(), ptr, ptr + sizeof(uint32_t));
  
  sendCustomData(custom_data);
  
  return hardware_interface::return_type::OK;
}
```

### 示例2：发送IMU数据

```cpp
void sendImuData()
{
  // 方式1：使用浮点数组（最简单）
  std::vector<float> imu_data = {
    static_cast<float>(imu_accel_[0]),  // ax
    static_cast<float>(imu_accel_[1]),  // ay
    static_cast<float>(imu_accel_[2]),  // az
    static_cast<float>(imu_gyro_[0]),   // gx
    static_cast<float>(imu_gyro_[1]),   // gy
    static_cast<float>(imu_gyro_[2]),   // gz
    static_cast<float>(imu_rpy_[0]),    // roll
    static_cast<float>(imu_rpy_[1]),    // pitch
    static_cast<float>(imu_rpy_[2])     // yaw
  };
  sendFloatArray(imu_data);
  
  // 方式2：使用字节数组（更灵活）
  uint8_t buffer[100];
  size_t offset = 0;
  
  // 添加时间戳
  uint32_t timestamp = static_cast<uint32_t>(
    std::chrono::system_clock::now().time_since_epoch().count() / 1000000
  );
  std::memcpy(buffer + offset, &timestamp, sizeof(uint32_t));
  offset += sizeof(uint32_t);
  
  // 添加加速度
  std::memcpy(buffer + offset, imu_accel_, 3 * sizeof(double));
  offset += 3 * sizeof(double);
  
  // 添加角速度
  std::memcpy(buffer + offset, imu_gyro_, 3 * sizeof(double));
  offset += 3 * sizeof(double);
  
  sendCustomData(buffer, offset);
}
```

### 示例3：发送机器人状态

```cpp
void sendRobotState()
{
  std::vector<uint8_t> state_data;
  
  // 1. 添加状态标识
  state_data.push_back(0x10);  // 状态包类型
  
  // 2. 添加所有关节位置
  for (size_t i = 0; i < joint_count; i++)
  {
    float pos = static_cast<float>(state_positions_[i]);
    uint8_t* ptr = reinterpret_cast<uint8_t*>(&pos);
    state_data.insert(state_data.end(), ptr, ptr + sizeof(float));
  }
  
  // 3. 添加所有关节速度
  for (size_t i = 0; i < joint_count; i++)
  {
    float vel = static_cast<float>(state_velocities_[i]);
    uint8_t* ptr = reinterpret_cast<uint8_t*>(&vel);
    state_data.insert(state_data.end(), ptr, ptr + sizeof(float));
  }
  
  // 4. 添加时间戳
  uint32_t timestamp = static_cast<uint32_t>(
    rclcpp::Clock().now().nanoseconds() / 1000000
  );
  uint8_t* ptr = reinterpret_cast<uint8_t*>(&timestamp);
  state_data.insert(state_data.end(), ptr, ptr + sizeof(uint32_t));
  
  sendCustomData(state_data);
}
```

## 下位机接收

下位机使用 `process_usb_data()` 函数自动接收和解析，数据会存储在 `trans_fdb_data` 中。

你需要在下位机的 `process_usb_data()` 函数中添加自己的解析逻辑：

```c
// 在 trans_task_new.c 的 process_usb_data() 函数中
if (received_crc == calculated_crc)
{
    // CRC校验通过，解析数据
    uint8_t *data = rx_buffer + 4;  // 跳过帧头和长度
    
    // 根据你的数据格式解析
    // 示例1：解析浮点数数组
    size_t float_count = expected_data_length / sizeof(float);
    float *float_data = (float*)data;
    for (size_t i = 0; i < float_count; i++)
    {
        // 使用 float_data[i]
    }
    
    // 示例2：解析混合数据
    size_t offset = 0;
    uint8_t type = data[offset++];
    
    if (type == 0x01)
    {
        // 解析类型1的数据
        float value1;
        memcpy(&value1, data + offset, sizeof(float));
        offset += sizeof(float);
        
        uint32_t value2;
        memcpy(&value2, data + offset, sizeof(uint32_t));
        offset += sizeof(uint32_t);
    }
}
```

## 数据长度限制

- 最大数据长度：65535字节（uint16_t最大值）
- 建议单次发送不超过512字节
- 如需发送大量数据，建议分包发送

## 注意事项

1. **字节序**：默认使用小端序（Little Endian），与下位机保持一致
2. **数据对齐**：使用 `__attribute__((packed))` 确保结构体无填充
3. **线程安全**：所有发送函数都是线程安全的
4. **错误处理**：函数内部会检查空指针和空数据
5. **日志输出**：使用 RCLCPP_DEBUG 输出调试信息，不影响性能

## 性能建议

1. 尽量批量发送数据，减少发送次数
2. 避免在高频循环中发送大量数据
3. 使用浮点数数组函数比手动构建更高效
4. 预分配缓冲区，避免频繁内存分配

## 调试技巧

### 打印发送的数据

```cpp
void printHex(const std::vector<uint8_t>& data)
{
  std::stringstream ss;
  ss << "Data (" << data.size() << " bytes): ";
  for (size_t i = 0; i < std::min(data.size(), size_t(20)); i++)
  {
    ss << std::hex << std::setw(2) << std::setfill('0') 
       << static_cast<int>(data[i]) << " ";
  }
  RCLCPP_INFO(rclcpp::get_logger("TideHardwareInterface"), "%s", ss.str().c_str());
}
```

## 完整示例

```cpp
// 在你的代码中使用
void myCustomSendFunction()
{
  // 示例：发送机器人完整状态
  std::vector<uint8_t> packet;
  
  // 包头：数据类型
  packet.push_back(0xFF);  // 自定义类型标识
  
  // 添加4个关节的位置和速度
  for (size_t i = 0; i < 4; i++)
  {
    float pos = static_cast<float>(state_positions_[i]);
    float vel = static_cast<float>(state_velocities_[i]);
    
    uint8_t* ptr = reinterpret_cast<uint8_t*>(&pos);
    packet.insert(packet.end(), ptr, ptr + sizeof(float));
    
    ptr = reinterpret_cast<uint8_t*>(&vel);
    packet.insert(packet.end(), ptr, ptr + sizeof(float));
  }
  
  // 添加IMU数据
  for (int i = 0; i < 3; i++)
  {
    float val = static_cast<float>(imu_rpy_[i]);
    uint8_t* ptr = reinterpret_cast<uint8_t*>(&val);
    packet.insert(packet.end(), ptr, ptr + sizeof(float));
  }
  
  // 发送
  sendCustomData(packet);
  
  RCLCPP_INFO(rclcpp::get_logger("TideHardwareInterface"), 
              "Sent robot state: %zu bytes", packet.size());
}
```
