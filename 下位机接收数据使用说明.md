# 下位机接收数据使用说明

## 功能概述

已添加全局缓冲区来存储接收到的数据，供外部函数使用。

## 新增的功能

### 1. 全局数据缓冲区

```c
// 在 trans_task_new.c 中
static uint8_t received_data_buffer[MOTOR_MAX_DATA_LEN];  // 存储接收到的数据
static uint16_t received_data_length = 0;                 // 数据长度
static uint8_t data_ready_flag = 0;                       // 数据就绪标志
```

### 2. 获取数据函数

```c
/**
 * @brief 获取接收到的数据
 * @param data 输出缓冲区
 * @param length 输出数据长度
 * @return 1=有新数据, 0=无新数据
 */
uint8_t get_received_data(uint8_t *data, uint16_t *length);

/**
 * @brief 检查是否有新数据
 * @return 1=有新数据, 0=无新数据
 */
uint8_t has_new_data(void);
```

## 使用方法

### 方法1：轮询检查新数据

```c
void your_task(void)
{
    uint8_t data_buffer[256];
    uint16_t data_length;
    
    // 检查是否有新数据
    if (has_new_data())
    {
        // 获取数据
        if (get_received_data(data_buffer, &data_length))
        {
            printf("Received %d bytes\r\n", data_length);
            
            // 解析数据
            parse_received_data(data_buffer, data_length);
        }
    }
}
```

### 方法2：直接获取数据

```c
void your_task(void)
```

## 数据解析示例

### 示例1：解析混合数据（类型 + 时间戳 + 关节 + 状态）

假设上位机发送：
- 类型(1字节) + 时间戳(4字节) + 4个关节数据(32字节) + 状态(2字节) = 39字节

```c
void parse_received_data(uint8_t *data, uint16_t length)
{
    if (length < 39)
    {
        printf("Data too short: %d bytes\r\n", length);
        return;
    }
    
    size_t offset = 0;
    
    // 1. 解析类型
    uint8_t packet_type = data[offset++];
    printf("Type: 0x%02X\r\n", packet_type);
    
    // 2. 解析时间戳
    uint32_t timestamp;
    memcpy(&timestamp, data + offset, sizeof(uint32_t));
    offset += sizeof(uint32_t);
    printf("Timestamp: %lu\r\n", timestamp);
    
    // 3. 解析4个关节的数据
    for (int i = 0; i < 4; i++)
    {
        float position, velocity;
        
        memcpy(&position, data + offset, sizeof(float));
        offset += sizeof(float);
        
        memcpy(&velocity, data + offset, sizeof(float));
        offset += sizeof(float);
        
        printf("Joint %d: pos=%.3f, vel=%.3f\r\n", i, position, velocity);
        
        // 使用数据控制电机
        // set_motor_position(i, position);
        // set_motor_velocity(i, velocity);
    }
    
    // 4. 解析状态标志
    uint16_t status;
    memcpy(&status, data + offset, sizeof(uint16_t));
    offset += sizeof(uint16_t);
    printf("Status: 0x%04X\r\n", status);
}
```

### 示例2：解析纯浮点数数组

假设上位机发送：8个浮点数（4个关节的位置和速度）= 32字节

```c
void parse_float_array(uint8_t *data, uint16_t length)
{
    // 计算浮点数个数
    size_t float_count = length / sizeof(float);
    printf("Received %zu floats\r\n", float_count);
    
    float *float_data = (float*)data;
    
    // 解析关节数据（假设是 位置0, 速度0, 位置1, 速度1, ...）
    for (size_t i = 0; i < float_count / 2; i++)
    {
        float position = float_data[i * 2];
        float velocity = float_data[i * 2 + 1];
        
        printf("Joint %zu: pos=%.3f, vel=%.3f\r\n", i, position, velocity);
        
        // 使用数据
        // set_motor_position(i, position);
        // set_motor_velocity(i, velocity);
    }
}
```

## 数据流程

```
上位机发送
    ↓
USB CDC 接收
    ↓
process_usb_data() 解析
    ↓ CRC32校验通过
memcpy(received_data_buffer, data, length)  ← 保存到全局缓冲区
data_ready_flag = 1                          ← 设置标志
    ↓
你的任务调用 get_received_data()
    ↓
memcpy(your_buffer, received_data_buffer, length)  ← 复制数据
data_ready_flag = 0                                 ← 清除标志
    ↓
解析和使用数据
```

## 注意事项

### 1. 数据覆盖问题

如果新数据到达时旧数据还没被读取，旧数据会被覆盖。

**解决方案：** 及时读取数据

```c
void your_task(void)
{
    // 高频率检查（建议每次循环都检查）
    if (has_new_data())
    {
        uint8_t data[256];
        uint16_t length;
        get_received_data(data, &length);
        // 立即处理
    }
}
```

### 2. 线程安全

当前实现没有使用互斥锁，如果在多线程环境中使用，需要添加保护。

**改进方案：**

```c
// 添加互斥锁
static osMutexId_t data_mutex;

void trans_task_init(void)
{
    // 创建互斥锁
    data_mutex = osMutexNew(NULL);
}

uint8_t get_received_data(uint8_t *data, uint16_t *length)
{
    osMutexAcquire(data_mutex, osWaitForever);
    
    if (data_ready_flag)
    {
        memcpy(data, received_data_buffer, received_data_length);
        *length = received_data_length;
        data_ready_flag = 0;
        
        osMutexRelease(data_mutex);
        return 1;
    }
    
    osMutexRelease(data_mutex);
    return 0;
}
```

### 3. 数据长度检查

始终检查数据长度，避免缓冲区溢出。

```c
uint8_t get_received_data(uint8_t *data, uint16_t *length)
{
    if (data_ready_flag)
    {
        // 检查缓冲区大小
        if (received_data_length > MOTOR_MAX_DATA_LEN)
        {
            printf("Error: data too large\r\n");
            data_ready_flag = 0;
            return 0;
        }
        
        memcpy(data, received_data_buffer, received_data_length);
        *length = received_data_length;
        data_ready_flag = 0;
        return 1;
    }
    return 0;
}
```

## 完整示例

### 在你的控制任务中使用

```c
void motor_control_task(void)
{
    static uint8_t data_buffer[256];
    static uint16_t data_length;
    
    while (1)
    {
        // 检查新数据
        if (get_received_data(data_buffer, &data_length))
        {
            // 根据数据长度判断类型
            if (data_length == 32)  // 8个浮点数
            {
                float *positions = (float*)data_buffer;
                
                // 更新4个电机
                for (int i = 0; i < 4; i++)
                {
                    float pos = positions[i * 2];
                    float vel = positions[i * 2 + 1];
                    
                    // 控制电机
                    motor_set_position(i, pos);
                    motor_set_velocity(i, vel);
                }
            }
            else if (data_length == 39)  // 混合数据
            {
                size_t offset = 0;
                
                uint8_t type = data_buffer[offset++];
                
                uint32_t timestamp;
                memcpy(&timestamp, data_buffer + offset, 4);
                offset += 4;
                
                // 解析关节数据
                for (int i = 0; i < 4; i++)
                {
                    float pos, vel;
                    memcpy(&pos, data_buffer + offset, 4);
                    offset += 4;
                    memcpy(&vel, data_buffer + offset, 4);
                    offset += 4;
                    
                    motor_set_position(i, pos);
                    motor_set_velocity(i, vel);
                }
                
                uint16_t status;
                memcpy(&status, data_buffer + offset, 2);
                
                // 处理状态
                if (status & 0x0001)
                {
                    // 正常运行
                }
            }
        }
        
        osDelay(1);  // 1ms延迟
    }
}
```

## 调试技巧

### 打印接收到的数据

```c
void print_hex_data(uint8_t *data, uint16_t length)
{
    printf("Data (%d bytes): ", length);
    for (int i = 0; i < length && i < 50; i++)
    {
        printf("%02X ", data[i]);
        if ((i + 1) % 16 == 0) printf("\r\n  ");
    }
    printf("\r\n");
}

// 使用
if (get_received_data(data_buffer, &data_length))
{
    print_hex_data(data_buffer, data_length);
}
```

### 统计接收频率

```c
static uint32_t last_receive_time = 0;
static uint32_t receive_count = 0;

if (get_received_data(data_buffer, &data_length))
{
    receive_count++;
    uint32_t now = dwt_get_time_ms();
    uint32_t dt = now - last_receive_time;
    
    printf("Received #%lu, dt=%lu ms (%.1f Hz)\r\n", 
           receive_count, dt, 1000.0f / dt);
    
    last_receive_time = now;
}
```
