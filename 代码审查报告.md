## 配置检查清单

### URDF/ros2_control 配置
```xml
<hardware>
  <plugin>tide_hw_interface/TideHardwareInterface</plugin>
  
  <!-- 电机串口（必须） -->
  <param name="serial_port">/dev/ttyUSB0</param>
  <param name="baud_rate">921600</param>
  
  <!-- IMU串口（必须，且必须与电机串口不同） -->
  <param name="imu_serial_port">/dev/ttyUSB1</param>
  <param name="imu_baud_rate">115200</param>
  
  <!-- 其他参数 -->
  <param name="enable_virtual_control">false</param>
  <param name="need_calibration">false</param>
  <param name="publish_imu_data">true</param>
  <param name="publish_rpy">true</param>
  <param name="imu_frame_id">imu_link</param>
</hardware>
```

### 串口权限
```bash
# 添加用户到dialout组
sudo usermod -a -G dialout $USER

# 或者直接修改权限
sudo chmod 666 /dev/ttyUSB0
sudo chmod 666 /dev/ttyUSB1
```
