# 自定义混合数据发送示例

## 功能说明

你可以发送任意组合的数据：
- ✅ 浮点数（float）
- ✅ 整数（int8, int16, int32, uint8, uint16, uint32）
- ✅ 字节数组
- ✅ 任意混合组合

所有数据会自动打包成：`[0xFF 0xAA][长度][你的数据][CRC32]`

## 方法1：使用 PacketBuilder（推荐，最简单）

### 示例1：发送 10个浮点数 + 2个整数

```cpp
#include "tide_hw_interface.hpp"

void sendMixedData()
{
  PacketBuilder builder;
  
  // 添加10个浮点数
  builder.addFloat(1.1f)
         .addFloat(2.2f)
         .addFloat(3.3f)
         .addFloat(4.4f)
         .addFloat(5.5f)
         .addFloat(6.6f)
         .addFloat(7.7f)
         .addFloat(8.8f)
         .addFloat(9.9f)
         .addFloat(10.0f);
  
  // 添加2个整数
  builder.addInt32(12345)
         .addInt32(67890);
  
  // 发送
  sendCustomData(builder.getData());
}
```

### 示例2：发送关节状态（浮点数 + 整数混合）

```cpp
hardware_interface::return_type TideHardwareInterface::write(
    const rclcpp::Time& /*time*/,
    const rclcpp::Duration& /*period*/)
{
  PacketBuilder builder;
  
  // 1. 添加数据包类型标识（uint8_t）
  builder.addByte(0x01);  // 类型1：关节状态
  
  // 2. 添加时间戳（uint32_t）
  uint32_t timestamp = static_cast<uint32_t>(
    std::chrono::system_clock::now().time_since_epoch().count() / 1000000
  );
  builder.addUInt32(timestamp);
  
  // 3. 添加关节数量（uint8_t）
  builder.addByte(static_cast<uint8_t>(joint_count));
  
  // 4. 添加所有关节位置（float）
  for (size_t i = 0; i < joint_count; i++)
  {
    builder.addFloat(static_cast<float>(state_positions_[i]));
  }
  
  // 5. 添加所有关节速度（float）
  for (size_t i = 0; i < joint_count; i++)
  {
    builder.addFloat(static_cast<float>(state_velocities_[i]));
  }
  
  // 6. 添加状态标志（uint16_t）
  builder.addUInt16(0x1234);
  
  // 发送
  sendCustomData(builder.getData());
  
  RCLCPP_DEBUG(rclcpp::get_logger("TideHardwareInterface"),
               "Sent joint state: %zu bytes", builder.size());
  
  return hardware_interface::return_type::OK;
}
```

### 示例3：发送IMU + 关节数据

```cpp
void sendRobotFullState()
{
  PacketBuilder builder;
  
  // 包头：数据类型
  builder.addByte(0x02);  // 类型2：完整状态
  
  // IMU数据（9个浮点数）
  builder.addFloat(static_cast<float>(imu_accel_[0]))   // ax
         .addFloat(static_cast<float>(imu_accel_[1]))   // ay
         .addFloat(static_cast<float>(imu_accel_[2]))   // az
         .addFloat(static_cast<float>(imu_gyro_[0]))    // gx
         .addFloat(static_cast<float>(imu_gyro_[1]))    // gy
         .addFloat(static_cast<float>(imu_gyro_[2]))    // gz
         .addFloat(static_cast<float>(imu_rpy_[0]))     // roll
         .addFloat(static_cast<float>(imu_rpy_[1]))     // pitch
         .addFloat(static_cast<float>(imu_rpy_[2]));    // yaw
  
  // 关节数据（假设4个关节）
  for (size_t i = 0; i < 4; i++)
  {
    builder.addFloat(static_cast<float>(state_positions_[i]))
           .addFloat(static_cast<float>(state_velocities_[i]));
  }
  
  // 错误码（uint32_t）
  builder.addUInt32(0);  // 无错误
  
  // 发送
  sendCustomData(builder.getData());
}
```

## 方法2：手动构建（更灵活）

### 示例1：手动添加混合数据

```cpp
void sendManualMixedData()
{
  std::vector<uint8_t> data;
  
  // 1. 添加浮点数
  float f1 = 1.23f;
  float f2 = 4.56f;
  uint8_t* ptr = reinterpret_cast<uint8_t*>(&f1);
  data.insert(data.end(), ptr, ptr + sizeof(float));
  ptr = reinterpret_cast<uint8_t*>(&f2);
  data.insert(data.end(), ptr, ptr + sizeof(float));
  
  // 2. 添加整数
  int32_t i1 = 12345;
  ptr = reinterpret_cast<uint8_t*>(&i1);
  data.insert(data.end(), ptr, ptr + sizeof(int32_t));
  
  // 3. 添加字节
  data.push_back(0xFF);
  
  // 发送
  sendCustomData(data);
}
```

### 示例2：使用 memcpy

```cpp
void sendWithMemcpy()
{
  uint8_t buffer[100];
  size_t offset = 0;
  
  // 添加10个浮点数
  float floats[10] = {1.0f, 2.0f, 3.0f, 4.0f, 5.0f, 
                      6.0f, 7.0f, 8.0f, 9.0f, 10.0f};
  std::memcpy(buffer + offset, floats, 10 * sizeof(float));
  offset += 10 * sizeof(float);
  
  // 添加2个整数
  int32_t ints[2] = {100, 200};
  std::memcpy(buffer + offset, ints, 2 * sizeof(int32_t));
  offset += 2 * sizeof(int32_t);
  
  // 发送
  sendCustomData(buffer, offset);
}
```

## 数据包格式

### 发送的数据（你构建的）
```
[类型(1)] [时间戳(4)] [关节数(1)] [位置1(4)] [速度1(4)] ... [状态(2)]
```

### 自动打包后的完整包
```
[0xFF][0xAA][长度(2)][你的数据(N)][CRC32(4)]
```

### 示例：发送 3个浮点数 + 1个整数

**你的数据：**
```
float: 1.23  → 4字节
float: 4.56  → 4字节
float: 7.89  → 4字节
int32: 12345 → 4字节
总计：16字节
```

**自动打包后：**
```
0xFF 0xAA          # 帧头 (2字节)
10 00              # 长度=16 (2字节，小端序)
A4 70 9D 3F        # 1.23 (4字节)
B8 1E 91 40        # 4.56 (4字节)
F6 28 FC 40        # 7.89 (4字节)
39 30 00 00        # 12345 (4字节)
XX XX XX XX        # CRC32 (4字节)
总计：24字节
```

if (received_crc == calculated_crc)
{
    uint8_t *data = rx_buffer + 4;  // 跳过帧头和长度
    size_t offset = 0;

## 常见问题

### Q1: 如何确定数据的字节序？
A: 默认使用小端序（Little Endian），与下位机保持一致。

### Q2: 浮点数精度会丢失吗？
A: 使用 float（4字节），精度约7位有效数字。如需更高精度，可以发送 double（8字节）。

### Q3: 最多可以发送多少数据？
A: 理论最大65535字节，建议单次不超过512字节。

### Q4: 如何发送 double？
```cpp
builder.addDouble(double value)
{
  uint8_t* ptr = reinterpret_cast<uint8_t*>(&value);
  data_.insert(data_.end(), ptr, ptr + sizeof(double));
  return *this;
}
```
